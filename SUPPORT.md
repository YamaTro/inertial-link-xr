# Support

InertialLink XR is a community Research Preview with no guaranteed response
time or fitness for production use.

- For setup questions, use GitHub Discussions if enabled, otherwise open a
  question issue without private data.
- For reproducible bugs using synthetic data, use the bug-report template.
- For vulnerabilities, do **not** open an issue; follow [SECURITY.md](SECURITY.md).
- For discomfort or a physical-safety incident, stop using the software. Seek
  appropriate professional help when needed; this repository cannot provide
  medical advice.

Never post pairing keys, packet captures from real trips, location histories,
health information, or identifiable passenger data.
