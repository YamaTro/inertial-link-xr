# 安全上の注意

InertialLink XRは実験段階の研究用ソフトウェアです。医療機器ではなく、乗り物酔いを予防・治療・軽減できることを実証したものでもありません。方向、強度、遅延が不適切な視覚運動は、不快感を悪化させる可能性があります。

## 必ず守ること

- **同乗者専用です。** 運転、自転車、歩行、立ったままの乗車、機械操作中には使用・設定しないでください。
- 着席し、適切にシートベルト等を使用し、周囲と非常口を妨げないでください。
- 吐き気、めまい、頭痛、眼精疲労、方向感覚の異常、発汗、不安、追跡の乱れが少しでも出たら直ちに中止してください。「慣れるまで我慢」させてはいけません。
- 他人に参加を強要せず、健康状態や位置情報の開示を求めないでください。個別の医学的判断は適切な専門家へ相談してください。

## 試験の順序

最初から公道や路線バスで試してはいけません。

1. 合成データと異常パケットによるオフライン試験
2. ゴーグルを装着せず、机上で軸・符号・停止操作を確認
3. 静止した室内でCamera/XR Originが動かないこと、追跡喪失時に中立へ戻ることを確認
4. エンジンを止めた停車車両で固定具を確認
5. 閉鎖・管理された環境で、同乗者・監視者・短時間・事前の中止基準を用意
6. 人を対象に公表可能な研究を行う場合は、説明と同意、プライバシー保護、適切な倫理審査、事故対応計画を準備

電話はエアバッグ、操作装置、視界、非常口を妨げず、衝突時にも飛ばない方法で固定してください。

## 異常時動作

時刻同期と3個の連続した正常パケットを受信するまで出力しません。250 msより古いデータ、100 msを超える未来データは拒否します。正常データが250 ms途絶えると250 msかけて中立へ戻り、合計500 msで出力ゼロになります。不正・再送パケットでこのタイマーを延長してはいけません。

この数値は工学的な防御値で、全員に安全・快適だと証明する値ではありません。変更には異常系試験と明示的な安全レビューが必要です。
