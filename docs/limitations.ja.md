# 制限事項

- Quest/OpenXR端末全体へ重ねるシステムオーバーレイではありません。導入したUnityアプリの指定コンテンツだけを動かします。
- CameraやXR Originは動かしません。車内でのゴーグル追跡が乱れる問題を、この通信だけで修復することもできません。
- スマホIMUから長時間安定した位置や車両軌跡を求めることはできません。加速度の積分は誤差が急速に蓄積します。
- 加速度には重力が含まれます。重力・線形加速度・姿勢は端末ごとのセンサーフュージョンによる推定値です。
- `TYPE_GAME_ROTATION_VECTOR`のヨー角は任意でドリフトします。地理的方位や車両の世界座標姿勢ではありません。
- センサー周期、バッチ処理、発熱、OSのバックグラウンド制限、Unityフレーム処理、Wi-Fi省電力により遅延が変わります。
- UDPは欠損、重複、順序入替、遅延が起きます。HMACは改ざんを検出しますが、データを暗号化しません。
- SDK 37以降を対象とするAndroidホストでは、`ACCESS_LOCAL_NETWORK`の拒否・取消によりLAN UDPが停止します。必ず安全側に停止し中立へ戻してください。v0.2送信アプリはSDK 35対象のため`INTERNET`だけを使用します。
- 取付具の緩み、向き、共振は測定を変えます。既定の取付方向以外を自動推定しません。
- 本プロジェクトは有効性試験を実施していません。人による差が大きく、物理的に正確に見える動きが快適とは限りません。

詳しくは[英語版](limitations.md)、[安全上の注意](safety.ja.md)、[脅威モデル](threat-model.md)を参照してください。
